# Инструкция по запуску проекта Hookah CRM

## Шаг 1: Настройка базы данных

Откройте файл `.env` и настройте подключение к базе данных:

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=hookah
DB_USERNAME=root
DB_PASSWORD=ваш_пароль
```

## Шаг 2: Создание базы данных

Создайте базу данных в MySQL:
```sql
CREATE DATABASE hookah;
```

Или используйте phpMyAdmin/MySQL Workbench для создания базы данных.

## Шаг 3: Запуск миграций

Выполните команду для создания таблиц в базе данных:
```bash
php artisan migrate
```

## Шаг 4: Установка npm зависимостей (опционально)

Если нужно компилировать CSS/JS:
```bash
npm install
npm run dev
```

## Шаг 5: Запуск сервера разработки

Запустите встроенный сервер Laravel:
```bash
php artisan serve
```

Проект будет доступен по адресу: http://localhost:8000

## Шаг 6: Регистрация первого пользователя

1. Откройте http://localhost:8000/register
2. Заполните форму регистрации
3. Выберите роль (Администратор или Директор)
4. После регистрации вы будете перенаправлены на главную страницу

## Полезные команды

- `php artisan migrate` - запуск миграций
- `php artisan migrate:refresh` - пересоздание всех таблиц (удалит данные!)
- `php artisan serve` - запуск сервера разработки
- `php artisan route:list` - список всех маршрутов
